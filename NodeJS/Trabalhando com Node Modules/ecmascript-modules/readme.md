# EcmaScript Modules - ESM

## Configurar packjson para esc modules
> usar o "type":"module",

```
{
  "name": "ecmascript-modules",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "type":"module",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```
